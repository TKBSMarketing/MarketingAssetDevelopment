# Foundations Tree Experts — Ad Copy Sets

Each ad has: Primary Text (what people read), Headline (bold below the image), Description (gray text below headline), and Creative Direction (what image/video to use).

The philosophy: **Sell the destination, not the plane ticket.** We're not selling tree removal — we're selling peace of mind, a safe home, a beautiful yard, the relief of finally handling that thing that's been bugging you for months.

## CREATIVE SPECS REFERENCE

| Placement | Ratio | Resolution | Max File Size |
|-----------|-------|------------|---------------|
| Feed Image | 4:5 | 1080x1350 | 30MB |
| Feed Video | 4:5 | 1080x1350 | 4GB |
| Stories/Reels | 9:16 | 1080x1920 | 4GB |
| Carousel | 1:1 | 1080x1080 | 30MB |

**Video specs:**
- Length: 5-15 seconds (14s sweet spot)
- Hook in first 2 seconds (MOTION, not a static frame)
- Always add captions (85%+ watch muted)
- CTA at midpoint and end
- .MP4 or .MOV, H.264 compression

**Safe zones for 9:16:**
- Top 250px: platform UI overlap (keep clear)
- Bottom 340px: CTA button overlap (keep clear)
- Center 60% of frame: safe for all text/logos
