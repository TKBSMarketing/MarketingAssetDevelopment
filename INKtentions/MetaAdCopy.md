## Recommended Testing Plan (Dynamic Creative)

**Structure: 3 primary texts / 3 headlines / 2 descriptions per dynamic ad**

**Phase 1 -- Angle Test (Week 1-2):**
Create one dynamic ad per angle. Each ad gets:
- 3 primary texts from that angle
- 3 headlines (mix angles -- Meta will find what works)
- 2 descriptions (mix angles)
- Same video asset

Let Meta optimize for link clicks to maka.games. Compare angle-level performance.

**Suggested Phase 1 builds:**

| Ad | Primary Texts | Headlines | Descriptions |
|----|---------------|-----------|--------------|
| Family | P1, P2, P3 | H1, H3, H4 | D1, D3 |
| Bluffing | P4, P5, P6 | H4, H5, H6 | D4, D6 |
| Social Proof | P7, P8, P9 | H7, H8, H9 | D7, D8 |
| Theme | P9, P10, P11 | H9, H10, H11 | D5, D9 |
| Quick Play | P12, P13, P1 | H12, H13, H1 | D11, D12 |

**Phase 2 -- Consolidate (Week 3-4):**
Take the top 2-3 performing angles. Swap in fresh primary texts and headlines from the pools above. Kill underperforming angles entirely. Pull breakdown reports to see which specific combos Meta favors.

**Phase 3 -- Scale (Week 5+):**
Build lookalike audiences from website visitors. Run winning combos. Rotate in new copy from pools to keep creative fresh.

| Phase | Dynamic Ads | Per Ad (P / H / D) | Goal |
|-------|-------------|---------------------|------|
| 1 | 5 (one per angle) | 3 / 3 / 2 | Find winning angle |
| 2 | 2-3 (top angles) | 3 / 3 / 2 | Optimize combos within winners |
| 3 | 1-2 (proven winners) | 3 / 3 / 2 | Scale spend |
